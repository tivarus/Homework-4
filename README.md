# Домашнее задание №4, Варибрус Т.В. ИУ4-32Б, Вариант 8 "Найти все петли"
Перед использованием программы отредактировать файл graph.txt.

    Пример файла graph.txt: 
    graph {
        0 -- 0;
        0 -- 4;
        4 -- 2;
        4 -- 4;
        5 -- 18;
    }
    
Программа находит все петли и выводит их на экран.
